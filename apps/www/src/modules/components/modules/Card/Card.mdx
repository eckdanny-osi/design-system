### Loading State

Many `Card` instances may require async data fetching to resolve before the first _meaningful_ render. To reduce the disorienting layout thrash on users, use a `Placeholder` component to frame the `Card` to its approximate final dimension.

`Placeholder` may be used as a temporary stand-in for either a single line of block flow text or a homogenous grid:

<LoadingCardExample />

```jsx
<Card>
  <CardHeader>
    {this.state.isLoading ? <Placeholder /> : <CardTitle>Fred Jones</CardTitle>}
  </CardHeader>
  <CardBody>
    {this.state.isLoading ? (
      <Placeholder grid cols={2} rows={3} />
    ) : (
      <FriendsList friends={this.data} />
    )}
  </CardBody>
</Card>
```

The `grid` option is provided as a convenience to the developer. If your particular use case requires `Placeholder` to be visually mapped to content blocks with the _highest_ conformity, please compose content placeholders within the `Grid` components to match your implementation.
