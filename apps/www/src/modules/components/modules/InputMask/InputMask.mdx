## Synopsis

Use a `InputMask` for a masked inputs. Default maskChar: '_'.

## Examples

Phone number InputMask:

<DemoCard>
    <PhoneInputMaskExample/>
</DemoCard>

```jsx
  <InputMask mask='(999) 999-99-99'/>
```
Use InputMask for any masked input fields:

<DemoCard>
    <CustomInputMaskExample/>
</DemoCard>

```jsx
   <InputMask mask='aaa-999' maskChar="#"/>
```
