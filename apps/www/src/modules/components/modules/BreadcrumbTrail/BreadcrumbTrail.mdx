## Synopsis

Use a `BreadcrumbTrail` for a breadcrumbs navigation.
 <Breadcrumbs/>

## Props

<PropTable docgen={Docgen.BreadcrumbTrail} />

## Example

The BreadcrumbTrail usage:

<Breadcrumbs/>

```jsx
const items = [
  { path: '/', title: 'Dashboard' },
  { path: '/components', title: 'Components' },
  { path: '#', title: 'Breadcrumbs' },
]

<BreadcrumbTrail items={items}/>
```

You can get a breadcrumbs with disabled links if you don't pass 'path' param.

<DisabledBreadcrumbs/>

```jsx
const items = [
  { title: 'Dashboard' },
  { title: 'Components' },
  { title: 'Breadcrumbs' },
]

<BreadcrumbTrail items={items}/>
```
