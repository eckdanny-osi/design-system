## Synopsis

`Button`s indicate an opportunity for users to trigger a system response when activated.

## Usage Guidelines

### Primary Buttons

- Primary buttons should be used to call attention to an action on a form, or highlight the strongest calls to action on a page, or in a card.
- Keep in mind that not every screen needs a primary button.

### Secondary Buttons

- Secondary buttons should be used for secondary actions, semi-destructive actions such as "Cancel," or to discourage usage.

### Buttons with Icon

- Buttons with Icon may be used when you want to draw more attention to the button, or when an icon helps to clarify or convey more meaning.
