## Examples

Alerts should not be used for field level validation - see invalid fields for this.

### Typical

<Badge color="info" className="mb-3">
  active
</Badge>

### Colors

<div className="mb-3">
  {["info", "success", "primary", "warning", "danger", "dark", "light"].map(
    d => (
      <Badge className="mr-1" key={d} color={d}>
        {d}
      </Badge>
    )
  )}
</div>

### Pills

<div className="mb-3">
  {["info", "success", "primary", "warning", "danger", "dark", "light"].map(
    d => (
      <Badge className="mr-1" pill key={d} color={d}>
        {d}
      </Badge>
    )
  )}
</div>

### Links

<div className="mb-3">
  {["info", "success", "primary", "warning", "danger", "dark", "light"].map(
    d => (
      <Badge className="mr-1" href="badge" key={d} color={d}>
        {d}
      </Badge>
    )
  )}
</div>
