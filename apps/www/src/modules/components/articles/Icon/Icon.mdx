## Usage Guidelines

The design system Iconography is most effective when used _consistently_!

- DO's
  - foo
  - bar
  - bizz
- DON'T
  - quo
  - qux

### Accessibility

The `Icon` renders an non-focusable `svg` element with the `[aria-role="hidden"]`. Provision of `alt` text is typically not required. Do ensure that if you are using an `Icon` within an interactive container that the parent element does have `alt` text and `aria-*` attributes as required.
