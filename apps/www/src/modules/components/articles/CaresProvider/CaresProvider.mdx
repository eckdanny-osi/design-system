## Synopsis

The `CaresProvider` component allows application developers to register user-defined Components on a React Context

```jsx
import { CaresProvider } from "@cwds/components";
```

## Examples

```jsx
import { CaresProvider, Page } from "@cwds/components";

// A User-Defined Component
const Brand = <a href="/">Acme Widgets</a>;

// Register `Brand` using `CaresProvider`
const App = () => (
  <CaresProvider Brand={Brand}>
    <Page
      title="My Awesome App"
      main={props => (
        <div>
          I am the <tt>Page</tt> body!
        </div>
      )}
    />
  </CaresProvider>
);
```
