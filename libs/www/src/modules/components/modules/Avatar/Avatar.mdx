## <a href="#Usage-Guidelines">Usage Guidelines</a>

Observe the following when using `Avatar`s:

* Display a default image if no custom image has been uploaded.
* `imgUrl` takes precedence over `text` display
* `text` takes precendence over the default icon

### <a href="#Using-Initials">Using Initials</a>

When user or identifiable entity does not have an `imgUrl` (path to a raster graphic), use the `text` prop instead:

<div className="mb-3">
  <Avatar text="DE" />
</div>

<SyntaxHighlighter language="jsx" style={xonokai}>
  {`<Avatar size="lg" text="DE" />`}
</SyntaxHighlighter>
