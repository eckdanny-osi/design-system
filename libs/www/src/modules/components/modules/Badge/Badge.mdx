## Synopsis

Use a `Badge` to enumerate the common classifications of a type of thing.

<DemoCard>
  <Badge color="primary">Badge</Badge>
</DemoCard>

<BadgePropdefs />

```jsx
import { Badge } from '@cwds/components';
```

* aslkdfjasdf

## Examples

Alerts should not be used for field level validation - see invalid fields for this.

### Typical

<Badge color="info" className="mb-3">
  active
</Badge>

### Colors

<div className="mb-3">
  {
    [
      'info', 'success', 'primary', 'warning', 'danger', 'dark', 'light'
    ].map(d => <Badge className="mr-1" key={d} color={d}>{d}</Badge>)
  }
</div>
